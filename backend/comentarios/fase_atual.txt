--------------------------------------------------------------------------------
  PRÓXIMO PASSO IMEDIATO
--------------------------------------------------------------------------------
 
Iniciar o frontend React.
 
O backend está pronto para o MVP. O frontend pode consumir a API.
Documentação dos endpoints: /docs (Swagger)
 
Fluxo de autenticação para o frontend:
  1. POST /auth/token → recebe access_token + primeiro_acesso
  2. Se primeiro_acesso == true → redirecionar para tela de troca de senha
  3. POST /auth/trocar-senha com token no header
  4. A cada response 200 autenticado → ler header X-Session-Expires e atualizar timer
  5. Se timer expirar sem atividade → fazer POST /auth/logout e redirecionar para login
  6. Se receber 401 → token expirado ou inválido → redirecionar para login
